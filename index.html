Pulsar

Smart and intuitive E-Commerce website search engine

Features

Pulsar is a powerful, easy to use tool that represents
and prioritise data based on relationships within dataset

Graphical representation

Pulsar uses D3.js library to provide powerful, and yet intelligible and easy tool for browsing and analysing search results

Relative results comparison

Pulsar not only returns results based on search parameters, but also helps you find the best ones by providing relative comparison for data

Adaptive search

Pulsar leverages adaptive ranking algorithm and provides comparison of results based on aggregated properties, giving you more flexibility in searching
