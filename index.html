Features

Pulsar is a powerful, easy to use tool that uses relationships
within data to display and prioritise results

Visualisation

Pulsar uses D3.js library to provide powerful, and yet intelligible and easy way for browsing and analysing search results.

Relative results comparison

Pulsar not only returns results based on search parameters, but also helps you find the best ones by performing relative data comparison.

Adaptive search

Pulsar leverages adaptive ranking algorithm and provides results comparison based on aggregated properties, giving you more flexibility in searching.
